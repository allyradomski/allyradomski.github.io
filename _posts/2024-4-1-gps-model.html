--- title: GPS-Tracking 3D Model description: Expressing data from walks media: /assets/journal/2024-4-26-expressive-data tags: - Mixed Media - 3D Print - Blender (Geometry Nodes) --- This project was a uni project about expressing data in different forms. I decided to track my freeform movement around a place I am fond of. I translated this GPS data into a Blender model using Geometry Nodes. I then exported this into a 3D model to be printed.
